## 🗂️ アーカイブポリシー

### アーカイブ基準
- 新バージョンリリース時の旧バージョンドキュメント
- 確認レポート・実装レポート（一時的なもの）
- 6ヶ月以上更新されていないドキュメント
- 技術的に obsolete になったドキュメント

### アーカイブ手順
1. `archive/` ディレクトリに移動
2. バージョン別フォルダに整理
3. アーカイブ理由をREADMEで記載

### 保持期間
- 直近3バージョンのドキュメント
- 重要な設計判断記録は永続保持
- 確認レポートは1年間保持
