## 📝 ドキュメント作成方法

### 1. 既存ドキュメントの貼り付け

以下のファイルに直接内容を貼り付けることができます：

- **仕様書**: `次期バージョン/仕様書.md`
- **設計書**: `次期バージョン/設計書.md`
- **要件定義**: `次期バージョン/要件定義.md`

### 2. 新しいドキュメントの作成

必要に応じて新しいファイルを作成してください：

```bash
# 例: 技術仕様書を作成
touch ドキュメント/active/next-version/次期バージョン/技術仕様書.md
```

### 3. ドキュメントの整理

- 大きなドキュメントは分割して管理することを推奨します
- 日付やバージョン番号をファイル名に含めることも可能です
- 例: `仕様書_v1.1_20251120.md`
