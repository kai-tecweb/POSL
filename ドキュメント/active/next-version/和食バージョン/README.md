## 📝 ドキュメント一覧

このディレクトリには、和食バージョンの開発に関するドキュメントを配置します。

### 準備済みファイル（貼り付け可能）

以下のファイルに直接内容を貼り付けることができます：

1. **`仕様書.md`** - 機能仕様、要件定義など
2. **`設計書.md`** - アーキテクチャ設計、データベース設計など
3. **`要件定義.md`** - ビジネス要件、ユーザー要件など

### その他のドキュメント

## 📋 ドキュメント作成ガイド

### 1. 既存ファイルへの貼り付け

既存のファイル（`仕様書.md`、`設計書.md`など）に直接内容を貼り付けてください。

### 2. 新しいファイルの作成

新しいドキュメントが必要な場合は、このディレクトリに新しい`.md`ファイルを作成してください。

### 3. ファイル命名規則

推奨される命名規則：

- 日本語ファイル名を使用可能
- 日付を含める場合: `仕様書_20251120.md`
- バージョンを含める場合: `設計書_v1.0.md`
- 複合: `仕様書_v1.0_20251120.md`
